# Portfolio Optimization for Mourabaha Financing

This project contains the source code associated with the research article: 
**"Managing Mourabaha Financing Risk via Quadratic Programming: A Strategic Asset Allocation Approach"**.

## Description
This work presents an approach based on quadratic programming to optimize asset allocation in a Mourabaha financing portfolio. The objective is to maximize the Sharpe ratio while adhering to strategic sectoral constraints.

## Project Structure
- `app.py`: The main script containing the optimization model.
- `requirements.txt`: List of necessary Python libraries.
- `data.xlsx`: Historical data used for the analysis.

## Usage
To run this model, ensure you have Python installed.

1. Install the dependencies: `pip install -r requirements.txt`
2. Execute the script: `python app.py`
